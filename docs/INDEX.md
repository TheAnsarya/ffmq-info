**Reference:** Function reference, labels, constants  
**Status:** Progress reports, campaign status  
**Management:** TODO lists, roadmaps, issues  
**Historical:** Session logs, completion reports, corrections

### By Audience

**New Users:**
1. Start with README.md
2. Read BUILD_QUICK_START.md or QUICK_START_GUIDE.md
3. Try MODDING_QUICK_REFERENCE.md for modding
4. Refer to FAQ.md for common questions

**Developers:**
1. Read CONTRIBUTING.md
2. Review ARCHITECTURE.md and BUILD_SYSTEM.md
3. Study technical/* documentation
4. Use FUNCTION_REFERENCE.md for code reference
5. Follow coding-standards.md

**Modders:**
1. Start with MODDING_GUIDE.md
2. Use ENEMY_EDITOR_GUIDE.md for enemy editing
3. Read BATTLE_DATA_PIPELINE.md for data workflow
4. Refer to graphics-quickstart.md for graphics
5. See TEXT_EDITING.md for text changes

**Researchers:**
1. Review technical/* analysis documents
2. Study ROM_DATA_MAP.md and RAM_MAP.md
3. Read BATTLE_SYSTEM.md and other system docs
4. Check UNREACHABLE_CODE_CATALOG.md
5. Review FUNCTION_REFERENCE.md
